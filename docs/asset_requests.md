# 🎨 Liste des Ressources Créatives (Assets) — Direction Lamborghini Luxury

Pour finaliser l'application avec un rendu "State-of-the-Art", voici les éléments spécifiques à demander au client ou à l'équipe créative. Ces éléments doivent suivre la ligne artistique **"Dark Luxury / Performance"** établie.

---

## 🏎️ 1. Identité Visuelle (Branding)
- **Logo "Master"** :
  - Format : Vectoriel (`.svg` ou `.ai`).
  - Style : Épuré, géométrique, évoquant la force et la fiabilité. Éviter les logos trop colorés ou enfantins. Un blason ou un monogramme "B2B" style constructeur automobile serait idéal.
  - Versions : Une version "Or" (métallique) et une version "Blanc Pur" sur fond transparent.
- **Slogan (Baseline)** :
  - Existant : *"Lève-toi et va, brobro"*
  - Recommandation : Garder ce slogan mais le typographier de manière impactante (Tout en majuscules, espacement de lettres large).

---

## 🎬 2. Backgrounds & Vidéos (Immersion)
- **Cinématique de Splash / Landing** :
  - Format : `.mp4` haute résolution (vertical 9:16) + Version optimisée `.webm`.
  - Durée : 5-8 secondes en boucle (loop).
  - Contenu : Des plans "macro" de haute qualité (style slow-motion) montrant :
    - Des mains d'artisans au travail (soudure, outils de précision).
    - Des vues aériennes de nuit d'Abidjan (Pont Alassane Ouattara éclairé).
    - Note : Le rendu doit être sombre, avec des touches de lumière dorée ou orange.
- **Vidéos d'Onboarding** :
  - 3 clips courts (3-4 sec) illustrant la Rapidité, la Confiance et la Qualité.

---

## 🎨 3. Panel de Couleurs (Charte Verrouillée)
Demander confirmation ou adaptation sur ces codes Hexadécimaux :
- **Background Principal** : `#0A0A0A` (Obsidienne).
- **Couleur Primaire (Branding)** : `#D4AF37` (Metallic Gold).
- **Couleur d'Action (CTA)** : `#FF8C00` (Giallo/Orange Lamborghini).
- **Statuts** :
  - Succès : `#00C853` (Emerald Green).
  - Erreur : `#FF1744` (Vibrant Red).

---

## 📸 4. Photographie (Imagerie Premium)
- **Portraits Artisans "Héros"** :
  - Portraits studio sur fond gris foncé ou noir.
  - Éclairage contrasté (style "Chiaroscuro") pour donner un aspect héroïque et pro aux artisans.
- **Icônes de Catégories Personnalisées** :
  - Au lieu de simples emojis, des icônes minimalistes en trait fin (Line Icons) ivoire ou or.

---

## ✨ 5. Micro-Animations (Lottie)
- **Animation de Recherche** : Un radar circulaire minimaliste style "Navigation Radar" de voiture de luxe.
- **Animation de Succès** : Une coche (checkmark) qui se dessine avec un éclat doré.
- **Indicateur de Chargement** : Une ligne cinétique qui rappelle le mouvement d'un tachymètre.

---

## 📂 Spécifications Techniques pour les Livrables
- **Qualité** : 4K pour les vidéos, 300 DPI pour les images.
- **Poids** : Les vidéos doivent être compressées intelligemment pour ne pas ralentir l'app sur des connexions 3G (Abidjan).
- **Transparence** : Tous les assets UI doivent être livrés sans fond (PNG 24 bits ou SVG).
